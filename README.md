# musicScrapper

CLI tool for scraping information from musical website (Rateyourmusic, Metal
Archives), with nice album ASCII art.

## Usage

```shell

musicScrapper [OPTIONS] "name_of_artist"

-credits
        Display RYM credits
  -website string
        Desired Website ('metallum' or 'rym')
```
